# My Ping Pong Game 🏓

A simple **two-player Ping Pong game** built in Python using the **Turtle** module. This game features dynamic ball movement, player-controlled paddles, and a live scoreboard.  

---

## Features

- **Two-player controls**:  
  - Right paddle: `Up` / `Down` arrows  
  - Left paddle: `W` / `S` keys  

- **Ball movement with adjustable speed**: Ball accelerates after paddle hits for more challenging gameplay.  

- **Scoreboard**: Tracks and updates player scores in real-time.  

- **Collision detection**: Ball bounces off paddles and screen edges accurately.  

---

## How to Run

1. Make sure you have **Python 3.x** installed.  
2. Save `ball.py`, `paddle.py`, `scoreboard.py`, and `main.py` (your main script) in the same folder.  
3. Run the game:

```bash
python main.py
